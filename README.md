# 🐍 Python Small Apps

## 📌 Author

Oskar Jasłowski

This is a **personal project** containing a collection of small Python applications built as independent mini-projects.

---

## 🎯 Project Overview

**Python Small Apps** is a set of standalone Python programs, each located in its own folder.

Every app focuses on a specific functionality such as GUI tools, utilities, or simple experiments. Some projects also include additional resources like text files, images, or configuration files.

---

## 🧩 Included Applications

### 📋 basic-to-do-list

Simple to-do list app with task persistence using a text file.

### 📅 calendar

Basic calendar-related functionality.

### 🧠 face_detector

Face detection app using Haar Cascade (OpenCV).

### 📈 graph-app

Graph plotting or visualization tool.

### 🐍 python-compilator

Simple Python code execution tool (mini compiler concept).

### ⛏️ python_mc_clone

Small Minecraft-inspired project with assets and configuration.

### 🖼️ simple-image-editor

Basic image editing functionality.

### ⏱️ stopwatch-app

Stopwatch implementation.

### 📝 text-editor

Simple text editor with file saving/loading.

### 🌦️ weather-app

Weather application with icons/assets for conditions.

### ▶️ yt-app

YouTube-related utility (likely downloader or viewer).

---

## 🖥️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/oskarjaslowski/python-small-apps.git
cd python-small-apps
```

2. Navigate to a selected app:

```bash
cd app-folder-name
```

3. Run the script:

```bash
python code.py
```
